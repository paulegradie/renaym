# Screenshots Directory

Place your Renaym application screenshots here.

## Required Files

Add the following screenshot files to this directory:

- `ai-parsing.png` - Screenshot showing AI-powered filename parsing
- `tmdb-matching.png` - Screenshot showing TMDB metadata matching
- `batch-rename.png` - Screenshot showing batch rename operations
- `subtitle-support.png` - Screenshot showing subtitle pairing feature

## Recommended Specifications

- **Format**: PNG (for UI screenshots with transparency)
- **Resolution**: 1920x1080 or higher
- **Aspect Ratio**: 16:9
- **File Size**: Optimize to <500KB per image if possible

## Tips

1. Capture screenshots at high resolution
2. Use consistent window sizes
3. Include window shadows for professional look
4. Highlight key features with annotations
5. Use tools like ShareX (Windows) or Kap (macOS)

See `DEMO_ASSETS_GUIDE.md` in the root directory for detailed instructions.
